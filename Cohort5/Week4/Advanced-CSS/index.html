Advanced CSS

In the Advanced classes section, you will:

  • review CSS classes, ID's and other selectors
  • learn about specificity
  • learn the basics of creating responsive layouts using Flexbox.
  • learn how to import custom fonts and icons
You will complete mini-challenges to test your learning and submit them via GitHub.

Fonts and Icons
In the Fonts and Icons section, you will
  • learn how to import custom fonts from Google Fonts
  • learn how to import icons from Font Awesome
Flexbox
In the Flexbox section, you will
  • use the Flexbox Playground to experiment with flexbox properties
    • Flexbox Playground
  • create a responsive mobile layout
  • test your learning with a small challenge!
